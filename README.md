## Theme

`codem-cli` supports light and dark terminal themes.

- Set `CODEM_THEME=light` or `CODEM_THEME=dark` to force a theme.
- If `CODEM_THEME` is not set, the CLI attempts to detect the terminal background from `COLORFGBG` or `TERM` and defaults to dark.
